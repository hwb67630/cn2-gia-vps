## 一、先说清楚：电信CN2 GIA到底是什么东西

很多人把CN2 GIA说得很玄乎，其实没那么复杂。它就是中国电信搞的一条"VIP通道"。

中国电信原来的骨干网叫163网（AS4134），全国人民都在用，晚高峰就像早高峰的三环，挤得你怀疑人生。后来电信建了一张新网，叫"下一代承载网"（CN2，AS4809），专门给重要业务用。CN2又分两个等级：

- **CN2 GT（General Transit）**：CN2的入门款，去程走CN2，回程可能走163，所以晚高峰还是会有波动
- **CN2 GIA（Global Internet Access）**：CN2里的旗舰，去程和回程**全程走59.43高速节点**，独立回国链路，不和普通用户抢带宽

打个比方，CN2 GIA就是你花钱买了条专用车道，旁边是挤成一锅粥的普通车道。你开车过去，别人堵着，你溜达着。这就是为什么晚高峰别人卡成PPT，你CN2 GIA的机器还能跑满带宽。

CN2 GIA的缺点也很明显——贵。IP transit价格据说能到120美元/Mbps，所以提供CN2 GIA的商家不多，套餐常年缺货也是常事。
